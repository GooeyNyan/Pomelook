## 安装依赖

```bash
yarn install
```

## 运行项目

```bash
# 运行 tsc，与 react-native cli start
yarn start

# 运行相应平台的 app
yarn android
# 或
yarn ios
```

## 注意事项

### android

摇一摇打开菜单，开启热更新，host server 等
